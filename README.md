# wsjs

A static, responsive website built using modern frontend technologies with SEO-ready structure and a customizable template.

## Features
- Static webpage structure for fast performance  
- Fully responsive layout across all devices  
- SEO-friendly configuration and markup  
- Ready-made template for rapid development  
- Built with **Next.js** and **Tailwind CSS**

## Tech Stack
- **Next.js** – React framework for hybrid static/SSR pages  
- **Tailwind CSS** – Utility-first CSS framework  

## Project Goals
- Deliver a clean and performant static website  
- Implement SEO best practices  
- Provide a flexible template for future page development
